Interface Design

I love creating and styling user interfaces for projects. It's the cornerstone of any project. My philosophy is that with user interfaces, less is more. UI shouldn't be cluttered, but features should not be hidden or removed. For creative UI design you need to find the perfect layout that makes the UI simple at a surface level that at the same time provides advanced functionality.

I've been practicing UI design for at least 10 years, improving significantly over that duration. My most recent UI projects that I am proud of are BrastinOS, pictured on the right, and Minimal Music player. BrastinOS is an application layer operating system shell that I have spent 2 years working on as a passion project. It has taught me a significant amount about designing interfaces that arent just restricted to a single program. I've learnt how to deal with customisation, colour theory, visual heirarchy and smooth, subtle and snappy UI animations.

Physics Bowling, 2018

3D Modelling, Lighting and Layout

I started learning 3D modelling in 2018 so I could create assets for my 3D projects. After creating Physics Bowling, I started to push my skills with blender and since then I am now familiar with the software. I've created numerous renders which have improved my ability to light, texture, and position objects in a scene.

Recently I have experimented with physics simulations to create assets and props such as blankets, cushions and pillows with much less effort. I've also begun to create animations and learn how to optimise models and materials for rendering. I can create my own textures, UV map them to objects, and I can create them in a variety of styles. I am also able to project live action video or images onto geometry to artificially introduce objects and lighting into the scene. I can create models for games, software and iconography, examples of which can be seen by clicking the 'more info' button below.
